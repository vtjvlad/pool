# Шрифт для номеров на шарах

Положите сюда файл:

`NotoSans-Regular.ttf`

Скачать: [Noto Sans Regular](https://github.com/googlefonts/noto-fonts/raw/main/hinted/ttf/NotoSans/NotoSans-Regular.ttf)

Если файла нет, worker попробует CDN; при блокировке CDN номера на шарах могут не отображаться (игра работает).
