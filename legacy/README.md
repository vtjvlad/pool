# legacy/

Архив устаревшего кода. **Не подключается** к dev/build через Vite.

## Содержимое

| Файл | Описание |
|---|---|
| `index.js` | Старый static HTTP-сервер (`node index.js`). Не резолвит npm-модули (gl-matrix). Заменён `npm run dev` (Vite). |
| `const-legacy.js` | Справочник вытесненных констант и функций с комментариями о бывшем применении. |

## Правила

- Целые устаревшие файлы → перемещать в `legacy/`
- Отдельные константы/функции → копировать в `const-legacy.js` с комментарием: **было / назначение / заменено**
- Новый код не должен импортировать файлы из этой папки
